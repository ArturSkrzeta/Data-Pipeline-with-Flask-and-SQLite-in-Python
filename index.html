Handling Data in Steps in Data Pipelines

Intro

Data Pipeline is a series of following after each other steps of data processing. Whithin the series each step delivers an output that is the input to the next step:

  1. Generating data in the Company A app.
  2. Requesting the data with API by Company B.
  3. Handling big data sets with Python in Company B.
  4. Inserting data into Company B database.

Features

App includes following features:

  • RESTful API
  • Flask
  • SQLite
